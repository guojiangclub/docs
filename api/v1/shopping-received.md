 # 收货
 
 ****
    
**简要描述：** 

确认收货，只有待发货状态下用户才能够进行确认收货动作

**请求URL：** 
- `https://demo-open-admin.ibrand.cc/api/shopping/order/received `
  
**请求方式：**

- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|order_no |是  |string |订单编号   |


 **返回示例**
 


``` 
{
  "status": true,
  "code": 200,
  "message": "",
  "data": []
}
```
