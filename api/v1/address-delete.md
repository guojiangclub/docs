 # 删除收货地址
 
 ****
     
**简要描述：** 

删除收货地址

**请求URL：** 
- `https://demo-open-admin.ibrand.cc/api/address/{id} `
  
**请求方式：**
- DELETE

- 登陆后才可请求，需 [认证参数](https://www.ibrand.cc/docs/api/v1/authentication "认证参数")


 **返回示例**

``` 
{
  "status": true,
  "code": 200,
  "message": "",
  "data": ""
}
```
