
# 创建二维码
 ****

**简要描述：**


- 创建二维码


**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/coupon/QRCode?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|cards |是  |array |  |

**参考：**
```
{
  "cards":{
		 "action_name":"QR_CARD",
		  "action_info":{
		  	"card":{
		  		"card_id":"poGxkwryU6NCx9KNdm7x6w526kqg"
			
		     	}
		  	}
	}	
	

}
```
