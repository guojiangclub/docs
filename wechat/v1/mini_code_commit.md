
# 上传小程序代码
 ****

**简要描述：**


- 为授权的小程序帐号上传小程序代码


**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/mini/code/commit?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|template_id |是  |string |  |
|ext_json |是  |string |  |
|user_version |是  |string |  |
|user_desc |是  |string |  |
