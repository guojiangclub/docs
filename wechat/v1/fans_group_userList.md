
# 获取标签下粉丝列表
 ****

**简要描述：**


- 微信公众号获取所有标签


**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/fans/group/userList?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|groupid |是  |int |  标签名ID|
|nextOpenId |否  |string | nextOpenId 可选 |
