
# 创建临时二维码
 ****

**简要描述：**


- 微信公众号二维码创建临时二维码


**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/qrcode/temporary?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|scene_id |是  |string |  |
|expire_seconds |否  |int | expire_seconds 过期时间可选 |
