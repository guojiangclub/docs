
# 移动粉丝到指定标签
 ****

**简要描述：**


- 微信公众号移动粉丝到指定标签


**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/fans/group/moveUsers?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|openids |是  |array |  |
|formid |是  |int |  原标签ID|
|toid |是  |int |  指定标签ID|
