
#上传永久图文消息
 ****

**简要描述：**


- 微信公众号素材上传永久图文消息



**路由：**

```
http://demo-wechat-platform.ibrand.cc/api/medias/remote/article?appid={appid}

```
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appid |是  |string |  |
|title |是  |string |  |
|thumb_media_id |是  |string |  |
|author |是  |string |  |
|digest|是  |string |  |
|content |是  |string |  |
|source_url |是  |string |  |


 详细参数请参考微信文档
